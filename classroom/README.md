# Classroom

Classroom examples and student projects for Raspberry Pi Pico/PicoW.

## Subfolders

| Folder | Description |
|---|---|
| [accesspoint/](accesspoint/README.md) | WiFi soft AP + DHCP server + HTTP server |
| [AI/](AI/README.md) | Genetic algorithm maze solver (student skeleton) |
| [altmain/](altmain/README.md) | App launcher menu for Pico Display |
| [chatGPT/](chatGPT/README.md) | OpenAI completions API wrapper for Pico W |
| [microkarel/](microkarel/README.md) | Karel-the-robot interpreter on Pico Display |
| [ntpclient/](ntpclient/README.md) | NTP time sync client |
| [prange/](prange/README.md) | Prime iterator teaching example |
| [pacman/](pacman/README.md) | Pac-Man clone for Pico Display |
| [tictactoe/](tictactoe/README.md) | Tic-Tac-Toe with AI opponent |

## Standalone Files

### test.py

Basic WiFi connectivity test for Pico W. Connects to a network and verifies the connection is live.
